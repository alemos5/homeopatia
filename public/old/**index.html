Análisis combinado
Es el resultado de la suma de los valores del  estudio de las variables matemáticas concurrentes identificadas hasta el momento. Como una repertorización se agrupan los remedios que más han cubierto dichas variables permitiendo un análisis más dinámico.
Utilizando las opciones que están en la botonera superior se podrá "filtrar" la o las variables que el usuario crea conveniente. (La experiencia indica que mejores resultados se obtienen con el conjunto de los datos).

Análisis desplegado
Indica una lista de remedios ordenados según la jerarquía del Rango de Semejanza Matemático (RSM) que determina el Algoritmo según los datos ingresados del paciente.

Se destacan tres variables a tomar en cuenta:
  • La Consonancia Vocal".
  • Los remedios "Clave" y la
  • "Secuencia".

Combined Analysis
is the result of the sum of the study values of the current mathematical variables identified so far. Like in a repertorization the remedies that cover most of those variables are grouped, allowing a more dynamic analysis.
Using the "filters" you can select the variables you think are best. (experience shows that the best results are when all the variables are used).


Extended Analysis
Indicates a list of remedies organized according to their hierarchy in the Mathematical Similarity Rank (MSR). This list determines the Algorithm based on the information entered on the patient.

3 Variables are highlighted
  • The "Consonance".
  • The "Key Remedies"
  • The "Sequence".
